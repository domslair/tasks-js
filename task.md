Library Management System (20 Marks)



A school library keeps a list of available books.



Requirements:

Create an array containing at least 8 book names.

Ask the user to choose one of the following options:

Issue Book

Return Book

Search Book

Show All Books

If the user selects Issue Book:

Ask for the book name.

Search the array.

If found, remove it from the array.

Display "Book Issued Successfully".

Otherwise display "Book Not Available".

If the user selects Return Book:

Ask for the book name.

Add it to the library.

If the user selects Search Book:

Display whether the book exists.

If the user selects Show All Books:

Display every available book using a loop.





Lucky Draw Game (20 Marks)



Create a lucky draw game.



Requirements:

Generate a random number between 1 and 20.

Give the player 5 chances.

Ask the player to guess the number.

If guessed correctly:

Display "Congratulations! You Won."

Stop the loop immediately.

If the player fails after 5 attempts:

Display "Better Luck Next Time."

Display the correct number.

Also display the number of attempts used.



School Bus Management System (20 Marks)



A school bus has 15 seats.



Requirements:

Store seat numbers in an array.

Ask the user to choose one of the following:

Book Seat

Cancel Booking

Show Available Seats

If booking:

Ask for the seat number.

If available, remove it from the available seats and add it to a booked seats array.

Otherwise display "Seat Already Booked."

If cancelling:

Ask for the seat number.

Remove it from the booked seats array.

Add it back to the available seats array.

Display the updated list of available and booked seats.
