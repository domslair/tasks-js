# JavaScript Assignment

This repository contains solutions for the following JavaScript assignment tasks.

## Task 1: Library Management System

### Requirements

- Create an array containing at least 8 book names.
- Provide the following options:
  - Issue Book
  - Return Book
  - Search Book
  - Show All Books
- Issue a book by removing it from the array if available.
- Return a book by adding it back to the array.
- Search for a book and display whether it exists.
- Display all available books using a loop.

---

## Task 2: Lucky Draw Game

### Requirements

- Generate a random number between 1 and 20.
- Give the player 5 chances to guess the number.
- If guessed correctly:
  - Display "Congratulations! You Won."
  - Stop the game.
- If all attempts are used:
  - Display "Better Luck Next Time."
  - Show the correct number.
  - Show the total attempts used.

---

## Task 3: School Bus Management System

### Requirements

- Store 15 seat numbers in an array.
- Provide the following options:
  - Book Seat
  - Cancel Booking
  - Show Available Seats
- Book a seat if available.
- Cancel a booked seat and make it available again.
- Display the updated list of available and booked seats.
